Guía para el Nombrado de Archivos Excel

El sistema identificará automáticamente al estudiante extrayendo únicamente los dígitos numéricos del nombre del archivo subido. A continuación se muestran ejemplos prácticos de cómo nombrarlos:

Nombres recomendados y válidos
  • 36077673.xlsx (Cédula exacta - Recomendado)
  • V-36077673.xlsx (Extrae: 36077673)
  • expediente_36077673.xlsx (Extrae: 36077673)
Nombres inválidos (No funcionarán)
  • NotasJuan.xlsx (Sin números: Cero dígitos)
  • 3607-76-73.xlsx (Guiones internos: Evitar)
  • Notas_12_34_56.xlsx (Dígitos incorrectos)

Guía: Notas Certificadas

info 1. ¿Qué son las Notas Certificadas?

Este módulo permite cargar archivos Excel individuales con el histórico oficial de calificaciones de un alumno (notas certificadas de años anteriores). Cada archivo se almacena vinculado al estudiante correspondiente, y puede descargarse o unificarse con otros expedientes en un solo archivo consolidado.

badge 2. Regla Clave: Nombre del Archivo = Cédula

El sistema extrae automáticamente la cédula del alumno a partir de los dígitos numéricos del nombre del archivo. El estudiante debe estar registrado previamente en Expedientes.

✓ Válidos 36077673.xlsx V-36077673.xlsx expediente_36077673.xlsx
✗ Inválidos NotasJuan.xlsx 3607-76-73.xlsx Notas_12_34.xlsx

cloud_upload 3. Cómo Cargar y Descargar

  • Arrastra el archivo .xlsx a la zona de carga, o haz clic en ella para seleccionarlo desde tu equipo.
  • Tras seleccionarlo, aparece el botón "Cargar Expediente". Presíona para subirlo sin recargar la página.
  • El expediente cargado aparece instantáneamente en la tabla inferior.
  • Usa el botón de descarga (icono de flecha) en cada fila para descargar el .xlsx original de ese alumno.

layers 4. Unificar y Descargar Todo

  • Haz clic en "Seleccionar" para activar el modo de selección múltiple.
  • Marca los expedientes que deseas combinar usando los checkboxes.
  • Haz clic en "Descargar y unir todo" para generar un solo archivo Excel con todas las hojas unificadas.
  • El archivo resultante se descarga automáticamente como Notas_Certificadas_Unificadas.xlsx.

bug_report 5. Log de Errores Técnicos

El botón flotante de insecto (🐛) en la esquina inferior izquierda abre un panel con el registro detallado de cualquier error que haya ocurrido durante la carga o unificación. Útil para diagnosticar problemas sin salir de la página.
