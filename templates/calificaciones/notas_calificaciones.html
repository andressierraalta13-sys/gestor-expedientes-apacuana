Guía: Carga de Calificaciones

checklist 1. Pasos Previos Obligatorios

  • Selecciona el Período Académico al que pertenecen las notas (ej: 2024-2025).
  • Selecciona el Año Cursante (1er año, 2do año, etc.) del grupo de alumnos.
  • Prepara tu archivo Excel: puedes subir un libro con 3 pestañas o pestañas específicas nombradas M1, M2 y M3 para cargar de forma masiva los lapsos 1, 2 y 3 respectivamente. Si subes una sola hoja que no coincida con estos nombres, el sistema la interpretará como Definitiva Directa.

Si aún no tienes períodos creados, haz clic en el enlace "Gestionar Períodos Académicos" para crear uno antes de subir el archivo.

table_chart 2. Formato del Archivo Excel

Cada pestaña activa debe seguir la estructura administrativa. Las columnas requeridas son:

CEDULA (Columna B) Cédula de identidad del alumno (se limpia a solo dígitos)
COLUMNAS D a M Notas de las 10 materias consecutivas correspondientes a la pestaña
M1 / M2 / M3 Hojas correspondientes a los Lapsos 1, 2 y 3 del periodo académico
DEFINITIVA DIRECTA Subida de una sola hoja que asocia las notas directamente a la nota final

download Puedes descargar el formato oficial multi-hoja usando el botón "Aquí tienes un formato para subir tus archivos".

event_repeat 3. Carga y Cálculo de Notas

  • El sistema mapea las notas de las hojas M1, M2 y M3 a los lapsos L1, L2 y L3 respectivamente.
  • La Definitiva se calcula de forma automatizada como el promedio de los tres lapsos cuando los tres están cargados completos.
  • El archivo de una sola hoja carga las notas directo como Definitivas Directas.
  • Si el alumno ya poseía notas registradas en dicho lapso/materia, estas se sobrescribirán automáticamente con los nuevos valores del Excel.

bar_chart 4. Interpretación del Reporte Final

  • Notas Cargadas: Registradas exitosamente por primera vez.
  • Notas Actualizadas: Ya existían y fueron sobreescritas con el nuevo valor.
  • Alumnos No Encontrados: La cédula en el Excel no corresponde a ningún estudiante en el sistema. Se recomienda crear el expediente primero.
  • Errores de Dato: Notas fuera del rango 0–20 o datos mal formateados. Se indica el número de fila del Excel.
